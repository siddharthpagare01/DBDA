## **Session 7: Git/GitHub **

**1. What is the primary purpose of Version Control Systems?**
A) To compile code
B) To track changes in source code over time
C) To execute programs
D) To design user interfaces
**Answer: B**
**Explanation:** VCS tracks changes to files, allowing multiple developers to collaborate while maintaining history.

**2. Which Git command initializes a new repository?**
A) git start
B) git init
C) git new
D) git create
**Answer: B**
**Explanation:** `git init` creates a new Git repository in the current directory.

**3. What does 'git clone' do?**
A) Creates a branch
B) Copies a remote repository to local machine
C) Deletes a repository
D) Merges branches
**Answer: B**
**Explanation:** `git clone` creates a local copy of a remote repository.

**4. Which command shows the current status of the working directory?**
A) git show
B) git status
C) git log
D) git diff
**Answer: B**
**Explanation:** `git status` displays the state of the working directory and staging area.

**5. What is GitHub primarily?**
A) A cloud storage service
B) A web-based Git repository hosting service
C) A programming language
D) An operating system
**Answer: B**
**Explanation:** GitHub provides hosting for Git repositories with additional collaboration features.

**6. Which command stages changes for commit?**
A) git save
B) git stage
C) git add
D) git commit
**Answer: C**
**Explanation:** `git add` moves changes from working directory to staging area.

**7. What does 'git push' do?**
A) Downloads changes from remote
B) Uploads local commits to remote repository
C) Creates a new branch
D) Deletes a repository
**Answer: B**
**Explanation:** `git push` uploads local repository content to a remote repository.

**8. Which command shows commit history?**
A) git history
B) git show
C) git log
D) git commits
**Answer: C**
**Explanation:** `git log` displays the commit history in chronological order.

**9. What is the purpose of 'git pull'?**
A) Upload changes
B) Fetch and merge changes from remote
C) Delete branches
D) Create tags
**Answer: B**
**Explanation:** `git pull` fetches from and integrates with another repository or local branch.

**10. Which command creates a new branch?**
A) git create-branch
B) git new-branch
C) git branch
D) git switch
**Answer: C**
**Explanation:** `git branch <branch-name>` creates a new branch.

**11. What does 'git merge' do?**
A) Compares two branches
B) Combines multiple sequences of commits
C) Deletes branches
D) Tags a commit
**Answer: B**
**Explanation:** `git merge` joins two or more development histories together.

**12. Which command discards changes in working directory?**
A) git undo
B) git reset
C) git checkout --
D) git clean
**Answer: C**
**Explanation:** `git checkout -- <file>` discards changes in working directory.

**13. What is a 'fork' in GitHub?**
A) A type of branch
B) A personal copy of someone else's repository
C) A commit message
D) A merge conflict
**Answer: B**
**Explanation:** Forking creates a personal copy of another user's repository to your account.

**14. Which command shows differences between commits?**
A) git compare
B) git diff
C) git show-diff
D) git changes
**Answer: B**
**Explanation:** `git diff` shows changes between commits, commit and working tree, etc.

**15. What does HEAD represent in Git?**
A) The main branch
B) The current branch reference
C) The remote repository
D) The staging area
**Answer: B**
**Explanation:** HEAD is a reference to the current branch or commit.
